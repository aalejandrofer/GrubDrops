Twitch's integrity check blocks pure-HTTP drops mining. The sidecar replays a real browser session on every gql call. Two ways to hand it your cookies — pick one.

recommended

Helper CLI / pre-built app

The helper opens your local browser, snapshots twitch.tv cookies, and pushes them to the miner. No DevTools, no copy-paste. Cookies never touch this page.

one-liner
go run ./cmd/dropsminer-helper twitch {{.AccountID}}
From the repo root. Add --miner URL to target a remote deployment, --password PW to skip the admin prompt.

No Go installed? Prebuilt binaries (.exe / .app / AppImage) ship from the same cmd/dropsminer-helper entry — see the repo README for download links.

fallback

Manual cookie paste

  1. Open twitch.tv in any browser and sign in.
  2. DevTools → Application → Cookies → https://www.twitch.tv.
  3. Copy at minimum auth-token. The optional cookies help Twitch treat the session as a returning user.
  4. Paste below.
