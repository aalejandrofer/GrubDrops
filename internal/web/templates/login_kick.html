Download the helper

recommended

A tiny app that reads your local browser's kick.com cookies and pushes them here — no DevTools, no copy-paste. Grab the build for your OS:

⬇ macOS (Apple Silicon) ⬇ macOS (Intel) ⬇ Windows ⬇ Linux
  1. Make sure you're signed in to kick.com in your everyday browser.
  2. Double-click the downloaded helper. It opens a console and walks you through it.
    macOS / Linux: first chmod +x the file. macOS Gatekeeper blocks unsigned downloads — right-click → Open → Open Anyway, or run xattr -d com.apple.quarantine <file> once.
  3. When prompted, paste this Account ID (then just press Enter through the rest):
account id
{{.AccountID}}
No password needed — your Account ID is the only credential. Miner URL defaults to this deployment; channels auto-discover from each campaign's game.

Manual cookie paste

fallback
  1. Open kick.com in any browser and sign in.
  2. DevTools → Application → Cookies → kick.com.
  3. Copy kick_session + XSRF-TOKEN (required), and session_token (needed to claim drops). cf_clearance optional.

Run from source

developers

one-liner
go run ./cmd/grubdrops-helper kick {{.AccountID}}
No password needed. Channels auto-discover from each campaign's game. Add --miner URL for a self-hosted deployment. (Optional: --channel STREAMER to pin specific channels.)
