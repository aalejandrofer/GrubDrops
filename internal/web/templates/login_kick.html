Authorize {{.DisplayName}}

Kick has no public OAuth. The sidecar replays your existing browser session via a chromedp tab. When discovery starts logging blocked by cloudflare, the cookies have gone stale — re-paste here.

recommended

Helper CLI / pre-built app

The helper opens your local browser, snapshots kick.com cookies (kick_session, XSRF-TOKEN, cf_clearance), and pushes them. No DevTools, no copy-paste.

one-liner
go run ./cmd/dropsminer-helper kick {{.AccountID}} --channel STREAMER
Replace STREAMER with the kick channel login. Add --miner URL for a remote deployment, --password PW to skip the prompt.

No Go installed? Prebuilt binaries (.exe / .app / AppImage) ship from the same cmd/dropsminer-helper entry.

fallback

Manual cookie paste

  1. Open kick.com in any browser and sign in.
  2. DevTools → Application → Cookies → kick.com.
  3. Copy kick_session and XSRF-TOKEN (required). cf_clearance optional — sidecar generates its own when needed.
  4. Paste below + the channel you want to mine.
